### 🎯 关于我

huh？在看我？那就简单介绍一下  
这里是**Slok·琐律**，也是Slok7565，称呼我为琐律/Slok就行  

我是一名在代码世界和教室课桌间切换的高中开发者。作为**XtremeWave**团队的核心成员，我的C#游戏模组开发是持续学习的过程——在调试中积累经验，在项目中磨练技能，作为强迫症，我的最终版代码一定是整洁且高效的，或许你该去[看看](https://github.com/Slok7565?tab=repositories)？

当不跟编译器较劲时：  
🎧 在网易云编排电子交响曲，把代码的节奏变成旋律  
🎮 B站上捕捉游戏的高光时刻，顺带分享点开发小技巧  
♟ 棋盘上推演策略，这比任何算法题都考验逻辑  
🏀 球场上实践团队协作，毕竟写代码也不能总当独行侠  

比起修复代码的“`bug`”，我更享受捕捉朋友情绪的“`异常`”，调试他们的心情——这比写代码成就感高多了
欢迎继续探索我的项目空间——愿这里的代码能给你带来新的启发！
